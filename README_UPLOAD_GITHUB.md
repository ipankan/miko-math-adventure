# Miko Math Adventure - GitHub Pages Ready (Audio v3)

File ini sudah siap diupload ke GitHub Pages.

## Isi
- index.html: game utama, termasuk aset gambar, favicon, backsound, dan suara Miko yang sudah tertanam.
- .nojekyll: agar GitHub Pages menyajikan file statis apa adanya.

## Perubahan Audio v3
- Backsound dibuat lebih terdengar namun tetap lembut untuk anak.
- Musik otomatis mengikuti halaman, mulai setelah sentuhan/tombol pertama.
- Saat tubuh Miko disentuh, Miko berkata: "Haloo, nama ku Miko!"
- Saat jawaban benar, Miko berkata: "Kamu hebat!"
- Saat jawaban salah, Miko berkata: "Ayo coba lagi!"
- Suara Miko ditambah efek chirp/meow lembut agar terasa lucu dan ceria.

## Cara Upload
Upload `index.html` dan `.nojekyll` ke root repository GitHub, lalu aktifkan Settings > Pages > Deploy from branch > main > /root.
